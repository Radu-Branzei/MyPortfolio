Ludo

Web reenaction of the classic board game Ludo.

About This Project

Full-stack app web application developed in Angular, with the use of ASP.NET web technologies for the backend server-side connection. Players can create rooms, join existing ones and play real-time turn-based games of the board game Ludo.

How to join a game

  • After entering the starting page, insert an username and choose one of the two options: Join Lobby or Create New Lobby.
  • Insert the ID of an existing lobby in the case of joining an existing one. (The ID will have to be personally transmitted by the host.)
  • In the case of a host, wait for more players to join and be ready, then press Start Game.
  • In the case of a guest, click Ready and wait for the host to start the game.

    • The multiplayer lobby system allows players to create or join game rooms, invite friends, and manage game settings before starting a match. Players can chat with each other in the lobby, view available game rooms, and join ongoing matches.

    How to play

  • Wait for your turn, you are unable to interact with the interface if it's another player's turn.
  • Roll the dice.
  • Click one of your pieces. It will move a corresponding amount of squares, according to the dice result.
  • Landing on another player's square will knock their piece to the very beginning.
  • The exact number of remaining squares will have to be rolled for a piece to reach the final spot, the triangle.
  • Once a player gets all their pieces inside the triangle, they have officially finished the game, allowing the others to continue until only one loser is left.

    • (NOTE: This was a team project consisting of two people with equal contributions. I take most of the responsibility for the frontend side, the responsive design and the connection with the objects created in the backend counterpart.)

    Key Features

    Fast Performance

    Angular single-page applications allow for short loading times.

    Turn-Based Gameplay

    Each player takes turns in a structured flow, ensuring fair play and strategy-driven decision making.

    Multiplayer Lobby

    Host or join live games in a multiplayer lobby, where players can start a new game.

    Parallel Games

    Thanks to socket-based communication, you can host or join multiple games at once.
